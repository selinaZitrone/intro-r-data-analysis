Introduction to R for Data Analysis

Workshop by Selina Baldauf

This workshop equips beginners with essential R skills for data processing and visualization. Every session is followed by hands-on exercises, and Day 4 is dedicated to applying what you’ve learned to your own research data. No prior programming experience required.

Before you start

Please complete the preparations before Day 1 to ensure a smooth workshop start.
